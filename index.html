Un grid simple con 3 columnas de ancho fijo de 200px cada una.

5. Grid con Áreas

Uso de grid-template-areas para crear un layout completo con encabezado, pie de página, contenido principal y barras laterales.

display: grid;
grid-template-columns: 1fr 3fr 1fr;
grid-template-rows: auto 1fr auto;
grid-template-areas: 
    "header header header"
    "sidebar contenido anuncios"
    "footer footer footer";
grid-gap: 15px;

6. Grid con Posicionamiento Explícito

Uso de grid-column y grid-row para posicionar elementos de forma específica en el grid.

display: grid;
grid-template-columns: repeat(4, 1fr);
grid-template-rows: repeat(3, 100px);
grid-gap: 10px;

.item-a {
    grid-column: 1 / 3; /* Desde línea 1 hasta línea 3 */
    grid-row: 1 / 2;    /* Desde línea 1 hasta línea 2 */
}

7. Grid con auto-fill vs auto-fit

Comparación entre auto-fill y auto-fit para crear layouts responsivos automáticos.

auto-fill: Mantiene el tamaño de las celdas y agrega columnas vacías

display: grid;
grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
grid-gap: 15px;

8. Grid con Alineamiento

Uso de propiedades de alineación para controlar la posición de elementos individuales dentro de sus celdas.

display: grid;
grid-template-columns: repeat(3, 1fr);
grid-gap: 15px;
justify-items: center; /* Alineación horizontal de todos los items */
align-items: center;   /* Alineación vertical de todos los items */

.item-start {
    justify-self: start; /* Alineación horizontal individual */
    align-self: start;   /* Alineación vertical individual */
}

9. Grid Responsivo con Media Queries

Adaptación del número de columnas según el ancho de la pantalla usando media queries.

display: grid;
grid-template-columns: repeat(4, 1fr);
grid-gap: 15px;

@media (max-width: 900px) {
    .grid-responsivo {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (max-width: 600px) {
    .grid-responsivo {
        grid-template-columns: 1fr;
    }
}

10. Grid Avanzado con Áreas Responsivas

Layout completo con grid-areas que cambia su disposición en pantallas pequeñas.

display: grid;
grid-template-columns: 200px 1fr 200px;
grid-template-rows: auto 1fr auto;
grid-template-areas: 
    "header header header"
    "nav    main   aside"
    "footer footer footer";
grid-gap: 15px;

/* Cambio a layout vertical en pantallas pequeñas */
@media (max-width: 768px) {
    .grid-avanzado {
        grid-template-columns: 1fr;
        grid-template-areas: 
            "header"
            "nav"
            "main"
            "aside"
            "footer";
    }
}
